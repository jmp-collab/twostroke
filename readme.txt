A dark, garage-inspired block theme for scooter and moped enthusiasts, tuning workshops, racing teams and riding clubs.

== Description ==

TwoStroke is a block theme built for the scooter and moped scene: builders documenting their projects, tuning workshops presenting their services, racing teams and riding clubs publishing their season.

It ships with ready-made block patterns for build stories, technical spec sheets, before/after comparisons, team rosters, event schedules, sponsor rows and workshop pages. A light "Workshop" style variation is included. The theme uses system fonts only, loads no external resources and collects no data.

== Frequently Asked Questions ==

= Does the theme load any external resources? =

No. TwoStroke uses system fonts and bundles all of its assets. Nothing is loaded from remote servers and no user data is collected.

= How do I build a landing page without a page title? =

Assign the "Landing (no title)" template to a page and compose it from the bundled patterns, starting with "Hero: Garage".

= Where do I find the patterns? =

In the block inserter under the pattern category "TwoStroke: Garage".
